# Data artifacts

Generated by Phase 0 scripts. Imported by the Replit Lenny Tank app.

- `guests/{slug}.json` ‚Äî character sheets (50 guests). See `index.ts:CharacterSheet`.
- `chunks/{slug}.json` ‚Äî embedded transcript chunks (~500 words each). See `index.ts:Chunk`.
- `scenarios.json` ‚Äî 25 curated scenario cards across 5 buckets. See `index.ts:ScenarioDeck`.

All chunks use Ollama `nomic-embed-text` (768-dim), generated locally.

## Regenerating

```bash
npx tsx scripts/chunk-transcripts.ts
npx tsx scripts/embed-chunks.ts          # requires `ollama serve` running locally
npx tsx scripts/build-character-sheets.ts # requires ANTHROPIC_API_KEY in .env
```

Each script is idempotent ‚Äî re-running skips already-completed work.
